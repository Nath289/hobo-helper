# hobo-helper

A Tampermonkey userscript that enhances your [HoboWars](https://hobowars.com) experience.

## What is Hobo Helper?

**Hobo Helper** is a userscript that adds quality-of-life improvements directly to the game's pages in your browser — no additional software required beyond the Tampermonkey browser extension. It provides features like at-a-glance stats, quick-action shortcuts, visual improvements, and more.

## Features Overview

Hobo Helper breaks its functionality into individual modules. Notable features include:
- **Bank Helper:** Manage bank savings and custom financial goals.
- **Drinks & Mixing System:** Tools for drinking, stat enhancement, and Mixer recipes.
- **Stat & Training Helpers:** Quality-of-life additions for University training and Tattoo Parlor planning.
- **Location Specific Helpers:** Enhancements for the Northern Fence, Wellness Clinic, Message Board, Kurtz Camp, Bernard's Mansion, Can Depo, and Recycling Bin.
- **Settings Dashboard:** Granular toggles to enable or disable features directly from the in-game preferences page.

For a complete list of all active modules natively supported, see our [Features Documentation](FEATURES.md). To see what's changed recently, check out the [Changelog](CHANGELOG.md).

## Installation

Hobo Helper runs inside your browser via the **Tampermonkey** extension.

1. **Install Tampermonkey:** Ensure the extension is installed for your browser (Chrome, Firefox, Safari, Edge, or Opera).
2. **Install the Script:** Click here to install the latest version: [hobo-helper-latest.user.js](https://github.com/Nath289/hobo-helper/raw/refs/heads/main/output/hobo-helper-latest.user.js).
3. Tampermonkey will prompt you to install or update the script. Confirm to finalize installation.

For a more detailed step-by-step guide, please refer to the [Installation Guide](INSTALL.md).

## Legality

The script has been confirmed as legal to use by the game's administration in its current form, as it does not breach the HoboWars Game Rules (no macros, auto-refreshers, or automated scripts). It does not play the game for you or trigger automated actions.

For detailed information on why it complies with game rules, see the [Introduction](INTRO.md).
